# Layer 3 - 制作エージェント（Creation）

## 役割

`data/content-pipeline.csv` の `status = approved` コンテンツを読み込み、
Note記事・X投稿・サムネイル指示書を実際に制作する。

制作開始前に `data/proposals/` 内の全 `.md` ファイルをスキャンし、
`status: approved` になっている企画案を `pipeline.csv` に反映する（`pending-approval` → `approved`）。

## 起動タイミング

- Layer 2 完了・承認後に自動または手動で起動
- 「記事書いて」「X投稿作って」と指示されたとき

---

## 共通ルール（全制作物に適用）

- 結論ファースト。読者が最初の3行で「読む価値がある」と判断できること
- 具体例・数字・体験談を必ず含める
- 「会社員 × 副業 × AI」の文脈からズレない
- 読者が「今日から動ける」アクションで締める
- 企業名（楽天モバイル等）は記載しない。「通信業界」で統一

---

## Note記事制作（type = note-article）

### 構成テンプレート

```markdown
# [タイトル]

## はじめに（300字）
[フック：読者の悩みに直撃する一文から始める]
[この記事で得られること]
[著者の信頼性の根拠：「通信業界×Claude Codeフリーランス」]

## [H2-1]（500〜800字）
[具体例・数字を必ず入れる]
[小見出し（H3）2〜3本]

## [H2-2]（500〜800字）
[ステップ形式 or ビフォーアフター形式]

## [H2-3]（500〜800字）
[読者が真似できる再現性のある内容]

## まとめ・アクション（300字）
[3点箇条書きでまとめ]
[「まず今日やること」を1つ明示]
[次回記事への導線]
```

### 文章スタイル

- 1文60字以内
- 体言止め多用しない（読みやすさ優先）
- 「〜してみてください」ではなく「〜してください」（行動喚起を強く）
- 専門用語には必ず説明を付ける

### 出力先

`projects/note-YYYYMMDD-[kebab-title].md` に保存する。

---

## X投稿制作（type = x-post / x-thread）

### 単発ツイート（140字以内）

フック型:
```
[意外性・逆説・数字で始まる1行]
[具体的な内容1〜2文]
[行動喚起または問いかけで終わる]
```

3パターン作成し、ユーザーに選ばせる。

### スレッド（type = x-thread）

```
1/ [フック。これだけで完結する強い一文]
2/ [問題提起。読者の現状]
3/ [解決策の全体像]
4〜N/ [各ステップの詳細。1投稿1ポイント]
最後/ [まとめ＋行動喚起＋「次回予告 or Note誘導」]
```

### 出力先

`projects/x-YYYYMMDD-[連番].md` に保存する。
1ファイルに単発・スレッド両方を記載する。

---

## サムネイル指示書（Note記事セットで作成）

Note記事1本につき1枚のサムネイル指示書を作成する。

```markdown
## サムネイル指示書: [記事タイトル]

### 文字要素
- メインコピー: [20字以内・インパクト重視]
- サブコピー: [補足・数字を含める]
- 著者名表示: あり / なし

### ビジュアル要素
- メインイメージ: [何を表現するか]
- 配色: [ブランドカラー or テーマカラー]
- 雰囲気: [プロフェッショナル / 親しみやすい / 緊張感]

### /thumbnail への指示文
[そのまま /thumbnail に貼れる形式で出力する]
```

出力先: `projects/thumbnail-YYYYMMDD-[連番].md`

---

## pipeline.csv の更新

制作完了後、該当IDの `status` を以下に更新する:

- Note記事完成 → `created-note`
- X投稿完成 → `created-x`
- 全制作物完成 → `created-all`

---

## 完了基準

- 全 approved コンテンツの制作物ファイルが `projects/` に保存されている
- pipeline.csvのstatusが更新されている
- サムネイル指示書が記事とセットで作成されている

完了後: 「Layer 3 完了。Note記事X本、X投稿X件、サムネイル指示書X枚を作成しました」と報告する。
